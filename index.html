What This Project Demonstrates
Core Docker & DevOps concepts covered in Task 4
📦

Containerization

The web server is packaged into a Docker container using a Dockerfile with an Nginx Alpine base image.

🔄

Container Lifecycle

Manage containers with build, run, stop, and remove commands. Docker Compose handles orchestration.

💓

Health Monitoring

Built-in HEALTHCHECK in the Dockerfile pings the server every 30 seconds to verify it's responding.

🔁

Auto Restart

Configured with restart: unless-stopped so the container recovers automatically from failures.

🗂️

Volume Mounting

HTML files are mounted as a volume so changes reflect instantly without rebuilding the image.

🌉

Bridge Networking

Container runs on a custom Docker bridge network, isolating it while exposing port 8080 to the host.

Docker Commands Used
Key commands to build, run and manage this container
Terminal Commands
$ docker-compose up --build Build image & start container
$ docker ps List running containers
$ docker logs codealpha-webserver View container logs
$ docker inspect codealpha-webserver Inspect container details
$ docker exec -it codealpha-webserver sh Enter container shell
$ docker-compose down Stop & remove container
